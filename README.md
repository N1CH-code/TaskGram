# TaskGram

## 📋 О проекте

TaskGram — это современная платформа для поиска работы, фриланса и микрозадач внутри Telegram.

**Основная валюта:** Рубли (₽)

**Целевая аудитория:** Россия

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- PostgreSQL 15+
- Redis 7+
- Telegram Bot Token (получить у @BotFather)

### 1. Клонирование и установка

```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

### 2. Настройка окружения

```bash
cp backend/.env backend/.env.local
# Отредактируйте .env.local, укажите TELEGRAM_BOT_TOKEN и DATABASE_URL
```

### 3. База данных

```bash
cd backend
npx prisma migrate dev
npx prisma db seed
```

### 4. Запуск

```bash
# Терминал 1: Backend
cd backend
npm run start:dev

# Терминал 2: Frontend
cd frontend
npm run dev
```

### 5. Docker (альтернатива)

```bash
docker compose -f docker/docker-compose.yml up -d
```

## 📱 Основные функции

### Роли пользователей
- 🔧 Исполнитель — ищет работу/заказы
- 💼 Работодатель — ищет исполнителей
- 🔄 И то и другое — совмещает роли

### Типы проектов
- 💼 **Работа** — постоянная занятость
- 🚀 **Фриланс** — разовые проекты
- ⚡ **Микрозадачи** — быстрые задания

### Проектные комнаты
После принятия отклика автоматически создаётся проектная комната с:
- 💬 Чатом
- 📎 Файлами
- ⭐ Отзывами
- 📋 Историей действий

### Тарифы

**Для исполнителей:**
- 🆓 Бесплатно — ограниченные отклики
- ⭐ Premium — 490 ₽/мес
- 💎 Premium Pro — 990 ₽/мес

**Для работодателей:**
- 🆓 Бесплатно — до 2 публикаций/мес
- 📢 Платные публикации — от 99 ₽
- 🏢 Подписка — 1 490 ₽/мес

## 🔒 Безопасность

- Авторизация через Telegram (проверка initData)
- JWT-токены с ограниченным сроком действия
- Контакты скрыты по умолчанию
- Рекомендации по безопасной передаче файлов
- Защита от повторного отклика
